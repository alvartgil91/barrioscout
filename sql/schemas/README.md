# BigQuery DDL Schemas

Place `.sql` files here with `CREATE TABLE IF NOT EXISTS` statements for each BigQuery table.

Naming convention: `<dataset>.<table>.sql`

Example:
- `barrioscout_raw.ministerio_transactions.sql`
- `barrioscout_raw.osm_pois.sql`
- `barrioscout_clean.transactions.sql`
- `barrioscout_analytics.neighbourhood_scores.sql`
